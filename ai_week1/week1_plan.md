Week 1 — Goal: Environment + Python for ML fundamentals

Daily schedule (7 days)

Day 1 — Environment & Python refresh (4–6h)
- Verify Python and venv. Run setup_week1.sh (or follow README).
- Brush up core Python: data structures, list/dict comprehensions, functions, OOP basics.
- Exercises: small scripts manipulating CSVs with pandas.

Day 2 — NumPy & Pandas (6–8h)
- NumPy arrays, broadcasting, vectorized ops.
- Pandas DataFrame ops: load, clean, groupby, merge, missing values.
- Mini task: exploratory data analysis on a sample CSV (describe, plots).

Day 3 — Data viz & EDA (5–7h)
- Matplotlib / Seaborn basics. Plot distributions, correlations.
- Feature engineering basics: scaling, encoding, simple transforms.

Day 4 — Intro to ML concepts (6–8h)
- Supervised vs unsupervised, train/val/test split, cross-validation, metrics.
- Scikit-learn: linear regression, logistic regression, decision trees.
- Hands-on: train a small classifier, evaluate, and report metrics.

Day 5 — Model pipelines & experimentation (6–8h)
- scikit-learn Pipelines, GridSearchCV, experiment tracking with simple logs or wandb.
- Reproduceable experiments: seeds, saving models with joblib.

Day 6 — Intro to deep learning & PyTorch basics (6–8h)
- Tensors, autograd, simple nn.Module, training loop.
- Hands-on: small MLP on a tabular dataset or MNIST subset.

Day 7 — Review & small project (8+h)
- Combine weeks work: EDA -> model -> eval -> report (README + notebook).
- Prepare short write-up and code to show in interviews.

Deliverables by end of Week 1:
- repo/ai_week1/ with plan, requirements, setup script, a notebook with EDA + a simple model,
- short README describing how to run experiments and reproduce results.
