AI Week 1 — Quick start

0) From repository root:
   bash ai_week1/setup_week1.sh

1) Activate virtualenv:
   source .venv/bin/activate

2) Start Jupyter:
   jupyter lab

3) Files in ai_week1/
   - requirements.txt  (packages list)
   - setup_week1.sh    (setup helper)
   - week1_plan.md     (daily plan)
   - src/              (starter python code)
   - notebooks/        (place notebooks here)

Notes:
- Installing torch may require a specific wheel for CUDA; follow https://pytorch.org if needed.
- To save progress: commit ai_week1/ to git and push.
